Que es Hexo?

Hexo es un versátil y potente framework para construir blog´s.
Tu escribes los post´s en lenguaje Markdown (u otros lenguajes) y Hexo genera archivos estáticos con temas muy atractivos en tan solo algunos segundos.

Requerimientos

La instalación de Hexo es muy sencilla. Solo necesitas tener instaladas las siguientes herramientas

Node.js
Git

Para iniciar

Crear un nuevo post

$ hexo new “My New Post”

Mas información: Escribiendo post´s

Ejecutar el servidor

$ hexo server

Este comando se ejecuta (desde consola) situado en la misma carpeta donde existe la instalación de Hexo.
Para verificar su correcto funcionamiento navegar en Prueba funcionamiento

Mas información: Ejecutando el servidor

Generar los archivos estáticos

$ hexo generate

Mas información: Generación

Despliegue en sitios remotos

$ hexo deploy
